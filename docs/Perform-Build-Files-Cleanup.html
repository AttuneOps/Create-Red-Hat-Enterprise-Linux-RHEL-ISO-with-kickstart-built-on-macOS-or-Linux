Step 1 - Clean Build Files

Deletes the build files and kickstart ISO.

Connect via ssh:

ssh {automationworkerlinuxuser}@{automationworkerlinuxnode}

Execute the following script: 

                
DIR="{automationWorkerBaseDir}/build-{newOsLinuxNode.fqn}"
if [ -d "${DIR}" ]
then
    rm -rf "${DIR}" && echo "Removed ${DIR}"
else
    echo "No directory to remove."
fi

ISO="{automationWorkerBaseDir}/kickstart_{newOsLinuxNode.fqn}.iso"
if [ -f "${ISO}" ]
then
    rm -fv "${ISO}"
else
    echo "No ISO to remove."
fi
